## 2. สิ่งที่ต้องทำต่อ / ขั้นตอนถัดไป (Next Steps)

### 🗄️ การอัปเดตข้อมูลบนฐานข้อมูลจริง (Supabase)
เพื่อความสมบูรณ์และถูกต้องของฐานข้อมูล ให้คัดลอก SQL ด้านล่างนี้ไปรันใน **Supabase SQL Editor** ของโครงการจริง เพื่ออัปเดตคำอธิบายภาษาไทยให้ตรงกัน:

```sql
UPDATE public.projects 
SET description = 'ระบบบันทึกประวัติและสอบเทียบเครื่องมือวัดของไซต์งานก่อสร้างทั้ง 17 แห่ง ใช้ตรวจสอบค่าใช้จ่าย ส่งออกเอกสาร PDF และทำเรื่องขอโอนย้ายอุปกรณ์ข้ามไซต์งานโดยมี Super Admin เป็นผู้อนุมัติ'
WHERE title = 'MEC CALIBRATION SYSTEM';

UPDATE public.projects 
SET description = 'ระบบติดตามแผนงานและงบประมาณไอทีรายปี (Capex & Opex) ในรูปแบบ Gantt Chart พร้อมใช้เป็นระบบวัดผลงาน (KPI) ของเจ้าหน้าที่ไอทีในทีม'
WHERE title = 'IT PROJECT DASHBOARD';

UPDATE public.projects 
SET description = 'ระบบแจ้งซ่อมคอมพิวเตอร์และช่วยเหลือปัญหาไอทีของพนักงาน 160 คน พร้อมระบบส่งแจ้งเตือนผ่าน LINE และอีเมลอัตโนมัติ'
WHERE title = 'IN-HOUSE HELP DESK';

UPDATE public.projects 
SET description = 'ระบบวิเคราะห์และสรุปยอดค่าใช้จ่ายการพิมพ์เอกสารแยกตามแผนก ช่วยควบคุมและลดค่าใช้จ่ายไอที (Opex) ขององค์กร'
WHERE title = 'PRINT COST DASHBOARD';

UPDATE public.projects 
SET description = 'ระบบบริหารและจัดสรรสิทธิ์โปรแกรมลิขสิทธิ์ของบริษัท (เช่น M365, AutoCAD, ZwCAD) ช่วยประหยัดค่าใช้จ่ายได้กว่า 1.12 ล้านบาทต่อปี'

UPDATE public.projects 
SET description = 'แดชบอร์ดสรุปค่าใช้จ่ายโครงการก่อสร้างทั้ง 17 ไซต์งานแบบ Real-time เพื่อให้ผู้บริหารตรวจสอบสถานะการเงินและควบคุมงบประมาณของแต่ละโครงการได้ทันที'
WHERE title = 'MEC PROJECT COST DASHBOARD';
```
